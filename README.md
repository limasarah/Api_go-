# API de Gerenciamento de Clientes (Go + Gorilla Mux)

Esta é uma API REST simples desenvolvida em linguagem Go para o gerenciamento de dados de clientes. O projeto utiliza o pacote `gorilla/mux` para roteamento e manipulação dos verbos HTTP.

## 🚀 Tecnologias Utilizadas

* [Go](https://golang.org/) (versão 1.18+)
* [Gorilla Mux](https://github.com/gorilla/mux) - Roteador HTTP potente para Go.

## 📋 Funcionalidades (CRUD)

A API permite realizar as seguintes operações:

- **Listar todos os clientes** (GET)
- **Buscar um cliente por ID** (GET)
- **Cadastrar um novo cliente** (POST)
- **Remover um cliente** (DELETE)
